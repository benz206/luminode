# luminode
Personalized Dorm Illumination System, made to integrate directly with Spotify and your devices using a simple ESP32 microcontroller.

## Usage
You need to do quite a few things to get this up and running.

### Hardware
- ESP32 
- WS2812B LED Strips 12V
- 12V Power Supply

### Software
- Arduino IDE
- FastLED Library
- WiFiClientSecure Library
- Ticker Library

### Setup
1. Clone the repository
2. Open the project in the Arduino IDE
